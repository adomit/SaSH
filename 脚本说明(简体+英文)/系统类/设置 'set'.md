## 设置 参数1:项目名称, 参数2:数值
更改介面上各项功能的状态



- 参数1:更改项目的名称
- 参数2:要更改的数值


> 所有参数都可使用变数

```
设置 快速战斗, 1  //所有开|关 功能 1是开 0是关
设置 加速, 14
设置 自动丢弃名单, ?肉|?水晶    //问号开头代表模糊匹配
设置 自动丢弃, true  //也可以使用true false 真 假

变数 A, 快速遇敌
变数 B, 1
设置 &A, &B

```
